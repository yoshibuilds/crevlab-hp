# 生活費シミュレーター

年収・雇用形態・居住地・家族構成を入力するだけで、推奨家賃・各生活費・貯蓄目標などを自動算出するWebツールです。

## 特徴

- 純粋な静的サイト（HTML + CSS + JavaScript 1ファイル構成）
- バックエンド不要、全計算をブラウザ内で完結
- Chart.js（CDN）による円グラフ可視化
- 全47都道府県対応の地域係数
- モバイルファーストのレスポンシブデザイン

## ファイル構成

```
life-budget/
  index.html   すべての実装を1ファイルに統合
  README.md    本ファイル
```

## ローカルでの確認方法

`index.html` をブラウザで直接開くだけで動作します。

```bash
# macOS / Linux
open projects/apps/life-budget/index.html

# Windows
start projects/apps/life-budget/index.html
```

## デプロイ手順

### GitHub Pages

1. このリポジトリを GitHub にプッシュ
2. Settings → Pages → Source を `main` ブランチ・`/docs` または `/(root)` に設定
3. `index.html` が公開される URL が発行される

### Vercel（推奨）

```bash
npm i -g vercel
cd projects/apps/life-budget
vercel
```

ウィザードに従うだけで自動デプロイされます。無料プランで利用可能。

### Netlify

`life-budget/` フォルダをそのまま Netlify のダッシュボードにドラッグ＆ドロップするだけで公開完了。

## 計算仕様

### 手取り換算（額面入力時）

| 年収帯      | 換算率 |
|-----------|------|
| ～300万    | ×0.80 |
| 300～600万 | ×0.78 |
| 600～1000万| ×0.75 |
| 1000万～  | ×0.72 |

フリーランスの場合は上記からさらに×0.95（税・社保の自己負担分）

### 地域係数（手取り月収に対する家賃割合）

| グループ | 都道府県 | 係数 |
|--------|---------|-----|
| 都市部A | 東京     | 0.33 |
| 都市部B | 大阪・神奈川・京都 | 0.29〜0.30 |
| 都市部C | 愛知・福岡・埼玉・千葉・兵庫 | 0.26〜0.28 |
| 地方都市 | 北海道・宮城・広島・石川・岡山等 | 0.23〜0.26 |
| その他地方 | 残り道県 | 0.18〜0.22 |

## 注意事項

本シミュレーターは概算値であり、実際の生活費は個人の状況により大きく異なります。
投資・借入の判断には専門家への相談を推奨します。
